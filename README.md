
# Against All Odds

Against All Odds is a Choose Your Own Adventure (CYOA) game developed by me. Players make decisions that shape the storyline and lead to different outcomes.

## Features
- Multiple story paths
- Choice-based gameplay
- Interactive decision-making
- Different endings

## How to Play
Open the game and follow the prompts to make choices. Each decision affects the outcome of the story.

- JavaScript
